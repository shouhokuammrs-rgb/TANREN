## コンセプト: 「鍛冶場(かじば)」

五輪書の「鍛錬」が由来のアプリ。世界観は**鉄を打つ鍛冶場**——暗がりの中に、熱した鉄の橙が灯る。
装飾ではなく集中のためのデザイン。トレ中の脳は疲れている前提で、迷いを生む要素をすべて削ぐ。

## デザイン原則(優先順)

1. **トレ中は片手・汗・低思考**: タップ領域は大きく、今やるべき1つだけが光る。実行画面とタイマーが最優先スクリーン
2. **数字はヒーロー**: 重量・レップ・残り秒数は画面の主役。等幅数字(tabular-nums)で桁ブレを防ぐ
3. **暗がりに炎**: ダーク基調+橙アクセントは維持。橙は「行動」と「熱(達成)」にのみ使い、乱用しない
4. **未達は失敗ではない**: 黄=調整中のトーンを守る。赤は破壊的操作(削除)専用
5. **祝う時は思い切り祝う**: PR達成演出だけは例外的に派手でよい(火花・鉄を打つ瞬間のイメージ)

## トークン指針

- 背景: 現行の濃紺〜黒のレイヤー構造を整理(base / raised / overlay の3段)
- アクセント: 溶鉄オレンジ(現行の橙を微調整可)。ホバー/押下の輝度変化を定義
- セマンティック: 達成=緑 / 調整中=黄 / 破壊=赤 / 情報=青系1色
- 角丸・余白・影: スケールを定数化し全画面で統一(現状の画面ごとのバラつきを解消)
- タイポグラフィ: 見出し・本文・数字の3役。数字はサイズjump大きめ

## 画面別の力点

| 画面 | 力点 |
|------|------|
| 実行画面+タイマー | 最重要。1m離れても残り秒が読める/次にやる1セットが常に明確 |
| ホーム | 「今日のトレを始める」が最短1タップ。人体図は眺めて楽しいレベルまで磨く |
| サマリー/PR演出 | 鍛冶のメタファーで祝う。スクショして人に見せたくなる画 |
| ウィザード | 進捗が見え、写真撮影に安心感(ローカル保存明記) |
| アイコン | 仮ダンベルピクトから「鉄・鍛」モチーフの正式版へ(ホーム画面で映える) |

## やらないこと

- ライトテーマ(利用者1名がダーク派のため)
- 過剰なオンボーディングアニメーション
- 情報密度を犠牲にする装飾
