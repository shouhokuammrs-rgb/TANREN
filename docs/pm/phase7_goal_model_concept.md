# Phase 7 構想: 部位別ゴールモデル(DEC-013候補)

> 採番注記: DEC-012は強調ローテーション(先行実装)に割り当てたため、本構想はDEC-013候補とする。

- 作成者: PM (Claude web) / 方針承認: Eiichi
- 日付: 2026-07-23
- 状態: 構想段階(筋力標準の調査前)。調査→デザイン探索→INS化の順で進める
- 背景: 「腕・肩は大きくしたくないが少し筋肉は欲しい」という要望から、固定の優先度係数ではなく部位ごとのゴールとの距離で扱いを決めるモデルへ発展させる

---

## 1. ユーザーから見た姿(UI/UX)

### 1-1. ゴール設定(初回セットアップ拡張 or 設定画面)
- 部位ごとに3段階プリセットから選ぶ: **引き締め / しっかり / がっつり**
- 各選択肢には以下を添える:
  - **レベル別イラスト**(部位ごとのシルエット3段階。Claude Designが探索)
  - **生活言語の説明文**(例: 「Tシャツの袖から出る腕にうっすら形がある」「半袖で明らかに鍛えているとわかる」)
  - **注記1行**: 「見え方は体脂肪率でも変わります」
- 裏の数値(目標e1RM)はいつでも直接編集可能(プリセットは入口であって縛りではない)

### 1-2. 日々の体験(おまかせ生成)
- ゴールが遠い部位(胸・背中・腹)は高頻度・フルメニューで出る
- ゴールに近い部位(腕・肩)は頻度・ボリュームが自然に下がっていく
- ユーザーが係数やセット数を手動調整する必要はない。**ゴール到達の結果として「たまに・軽く」が実現される**

### 1-3. 成長ビューとの統合
- e1RM推移グラフに**ゴールラインを表示**(あとどれくらいかが見える)
- 人体図・部位カードにゴール達成度の表現を追加(具体案はデザイン探索)

### 1-4. ゴール到達チェック(好み問題への答え)
- 部位がゴールに到達したら通知: 「腕がゴールに到達しました。鏡を見て判定してください」
- 選択肢: **満足 → 維持モードへ移行** / **物足りない → ゴールを1段引き上げ**
- ゴールは「理想の見た目の約束」ではなく「**鏡を見るチェックポイント**」として扱う。見た目の好みの個人差は数値定義ではなく本人の目に委ねる
- 維持モード: 優先度低・種目1つ・セット数抑制で出現は続く(消えない)

## 2. 裏側の仕組み(エンジン・データ)

### 2-1. ゴールの内部表現
- ゴールは固定kgではなく **体重×係数** で保持(例: `arms: { level: 'toned', coef: 0.25 }`)
- 評価時に body_stats の最新体重で目標e1RMを再計算 → **減量・増量にゴールが自動追従**
- 係数は筋力標準(体重比・性別・年齢)の調査結果から3レベルに対応づける(§3調査項目)

### 2-2. 達成度とギャップ
- 現在地 = DEC-011の**基準種目e1RM**(部位ごと・growth.tsの既存純関数を再利用)
- 達成度 = 現在e1RM ÷ 目標e1RM(部位ごと)
- **腹の例外**: e1RMと相性が悪いためレップ数ベース or 対象外(調査項目)

### 2-3. 動的優先度(F-03の置き換え)
- 現行: 目標タイプ+want/avoidタグによる固定係数(WANT_BOOST / AVOID_FACTOR)
- 新: **優先度 = f(ギャップ)**。ゴールが遠いほど高く、近づくほど下がり、到達後は維持係数に収束
- 選択式は現行の `フレッシュネス × 優先度` を維持(selection.tsの構造は不変)
- injury(痛みフラグのハード除外含む)は現行どおり併存

### 2-4. 維持モードの生成調整
- 種目数1・セット数-1に抑制(menu.tsのボリューム調整に維持フラグを追加)
- 進行: 増量提案は継続するが2ステップジャンプ(余裕あり)は適用しない——ゆっくり強くなるのはOK、ボリュームで大きくしない

### 2-5. データモデル
- Goalテーブル拡張: 部位ごとの `{ level, coef, mode: 'growth' | 'maintain' }`
- 到達チェックの記録(いつ・どう判定したか)を残し、SSOT的に追跡可能にする

## 4. 決定済み事項(本ドキュメントの前提)

- 方式A(3段階プリセット+裏数値編集可)を採用
- ゴールは体重連動(体重×係数)とする
- ゴール到達時は鏡チェック→維持移行 or 引き上げのループとする
- 「鍛えたくない=除外」ではなく「ゴール到達で自然に落ち着く」を基本思想とする
